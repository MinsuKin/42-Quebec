# ft_printf : 가변인자를 활용한 printf 함수 만들기

## 배우는 것

> printf 와 매우 비슷한 출력을 하는 ft_printf 함수를 만드는 과제입니다.
시간이 오래걸리더라도 보너스까지 해보시는 것을 추천드립니다. 
구조를 짤 때, 처음에는 타인에게 힌트를 받지 말고 스스로 혼자 충분히 고민해봅시다. 끝까지 갔는데 전체적인 구조가 틀린 것 같으면 처음부터 다시 해봐도 됩니다.
구조도 잘 짜보고, 내가 의도한대로 로직이 잘 돌아가는지 디버거를 사용해봅시다. 디버거를 사용할 줄 모른다면 익혀봅시다. lldb도 좋고 vs code의 디버거도 좋습니다. 설정과 사용법을 충분히 익히고 사용해봅시다.
