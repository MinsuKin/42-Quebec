# so_long : 나만의 2D 게임 만들기

## 배우는 것

- 외부 라이브러리(mlx) 사용법 이해하고 사용하기
- 동적 라이브러리 이해
- makefile 플래그
- 게임 테마 기획
- 로딩 시간 최소화를 위한 설계
- 철저한 예외처리를 포함한 파일 읽기

##
> make

> ./so_long map.ber
